This styleguide includes components, typographical conventions, etc. The mark-up uses Zurb Foundation 4 conventions by default, but feel free to change it if you have your own (or another) CSS framework and/or conventions.

See Zurb's own Kitchen Sink page, for the most up-to-date and complete set of all of their Foundation components.

Every front-end developer has that perfect combination of tools and libraries that he or she already uses to improve their workflow. Sometimes its that special combination of plugins to trick out their favorite editor or IDE (e.g. the perfect Vim, Sublime Text, or WebStorm set up). Tables

Zurb Foundation tables default to being pre-styled with zebra-striping, borders, etc. If you don't want these you'll need to override the settings that get used by the scss/foundation/components/_tables.scss partial by defining the appropriate variables in your settings override (that gets included before foundation):

$table-bg: #fff !default;
$table-even-row-bg: #f9f9f9 !default;// e.g. make the same as $table-bg to remove striping
$table-border-style: solid !default; // e.g. change to 'none'
